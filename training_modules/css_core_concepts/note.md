### Short and Longhand Properties

```css
/* Shorthand properties are defined with multiple values. */
.class {
    padding: 10px 5px 20px 15px;
}

/* Longhand properties are defined with individual values. */
.class {
    padding-top: 10px;
    padding-right: 5px;
    padding-bottom: 20px;
    padding-left: 15px;
}
```
